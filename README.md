## 玩法
控制蛇`@+++`吃果子`o`

<kbd>W</kbd><kbd>S</kbd><kbd>A</kbd><kbd>D</kbd> 分别控制上下左右

<kbd>E</kbd> 确认选择

<kbd>Q</kbd> 返回上一步/暂停游戏
